7. Sorting, Filtering and Data Validation (বাংলা)

This video shows how to sort data by one or more columns, filter data based on criteria, and use data validation to restrict data entry.

This video provides a practical guide on organizing and protecting data in spreadsheets:

  • Sorting and Filtering (0:01 - 2:48): The video demonstrates how to sort data by numbers, text, or color, and how to use filters to manage both tables and standard cell ranges efficiently.
  • Data Validation (2:52 - 6:22): The tutorial explains how to use data validation to control user input—such as restricting cells to specific numeric ranges (e.g., 0-31 days) or enforcing minimum text character lengths—to ensure data integrity and prevent entry errors.
