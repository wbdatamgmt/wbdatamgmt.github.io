4. Module 4 - 4 - Advanced Question Types in Google Forms (বাংলা)

This video provides a tutorial on using three advanced question types to improve form design:

  • Multiple Choice Grid (0:43 - 3:13): Explains how to compress multiple questions sharing the same response options into a single, compact grid format.
  • Tick Box Grid (3:13 - 4:21): Demonstrates how to use this for ranking or preference questions where users can select multiple options, including settings to limit responses per column.
  • Linear Scale (7:28 - 8:36): Shows how to create rating-style questions, such as satisfaction surveys, using a customizable numerical scale (e.g., 1-5 or 1-10).
