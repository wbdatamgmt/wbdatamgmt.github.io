Introduction to Excel Assessment

This assessment is optional but highly recommended. Javascript-based answer checking will be added later.

  1. Which of the following mathematical functions in Excel calculates the total of a range of cells? a. AVERAGE b. COUNT X. SUM d. MAX/MIN

  2. How can you adjust the width of a column in Excel? Select all that are correct. X. Right-click on the column header and select “Format Column Width” X. Use the “Adjust Column Width” button on the Home tab X. Drag the boundary between two column headers to resize it d. Use the “Row Height” option in the Format Cells dialog box

  3. What does the “Paste Special” feature in Excel allow you to do? a. Paste only the formulas from copied cells b. Paste the formatting and styles from copied cells c. Paste comments and hyperlinks from copied cells X. All of the above

  4. How can you add a new sheet to an existing Excel workbook? Select all that are correct. X. Click on the “+” button next to the sheet tabs X. Right-click on any sheet tab and select “Insert Sheet” c. Use the “New Sheet” option in the File menu d. Press the “Ctrl + N” keyboard shortcut

  5. Which function can be used to combine the contents of two or more cells into one cell in Excel? Select all that are correct. X. AND X. CONCAT c. SPLIT d. TRIM

  6. To apply a border around a cell or range of cells in Excel, you should: a. Use the “Bold” option from the Font group on the Home tab b. Use the “Underline” option from the Font group on the Home tab X. Use the “Borders” option from the Font group on the Home tab d. Use the “Italic” option from the Font group on the Home tab

  7. What is the purpose of data validation in Excel? a. To automatically calculate values in cells b. To check for spelling errors in the spreadsheet X. To restrict the type of data that can be entered into a cell d. To create pivot tables for data analysis

  8. How can you sort data in Excel? Select all that are correct. X. By using the “Sort” button in the Data tab b. By dragging and dropping cells to rearrange them X. By using the “Sort & Filter” button on the Home tab d. By using the “Find and Replace” feature

  9. What advantage does using a Table provide in Excel? a. It allows you to perform complex calculations using advanced formulas. X. It automatically updates the formulas whenever data in the Table changes. c. It enables you to insert charts directly from the Table data. d. It provides access to exclusive Excel functions not available elsewhere.

  10. What is the purpose of using clear and concise labels in Excel spreadsheets? a. To make the data look more attractive b. To reduce the file size of the spreadsheet c. To prevent data loss X. To make the data easier to understand and interpret
