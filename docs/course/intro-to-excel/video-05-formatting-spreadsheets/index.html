5. Formatting Spreadsheets

This video focuses on techniques for formatting spreadsheets to make data more presentable and easier to analyze, using cell borders, colours, and styles.

Key topics covered include:

  • Clearing and Applying Styles: How to remove existing formatting and apply new colors and borders (0:47-2:25).
  • Number Formatting: Adjusting settings to display data in specific currency formats (e.g., Rupees) and managing decimal places (2:29-4:52).
  • Conditional Formatting: Using rules to automatically highlight data based on criteria, such as identifying employees who have exceeded a specific number of leave days (4:53-7:00).
