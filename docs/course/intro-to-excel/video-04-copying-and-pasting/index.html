Copying and Pasting

Jun 6, 2023

This video shows how to copy and paste data within a worksheet, between worksheets, and between workbooks, including Paste Special options.

While it introduces standard copy-paste shortcuts (Ctrl+C and Ctrl+V), it focuses primarily on the advanced Paste Special options available to users (1:16).

Key features covered include:

  • Paste Values: Allows you to copy only the cell content without retaining formatting like bold text, colors, or text wrapping (1:36).
  • Transpose: A helpful tool for flipping your data so that column headers become row headers (2:45).
  • Paste Formulas: Enables users to copy only the underlying formulas (indicated by the FX icon) from one cell to another while automatically adjusting the cell references (4:36).
  • Format Painter: Demonstrates how to quickly copy the aesthetic styling (like font weight and cell color) from one area to another without using standard clipboard commands (5:17).
