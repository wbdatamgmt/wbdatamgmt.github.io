Basics of Data Management

Basics of Data Management is a video-based course with 5 modules. Each module consists of 5-10 videos.

All the videos in the modules are free to access and participants can view them in any order they like. It is recommended to go in sequential order. Each video in the module is accompanied by an assignment. At the end of each module, there is a Module Completion Assessment. These are optional but highly recommended.
