Conditional Formatting

Jun 6, 2023

This video shows how to use conditional formatting to highlight important data, including colour scales and data bars.

This video provides a tutorial on using conditional formatting to analyze and visualize datasets (0:00-12:00). The video demonstrates how to:

  • Apply Custom Rules: Using formulas to compare columns, such as identifying schools with more girls than boys (1:40-7:07).
  • Use Fixed vs. Variable References: Explaining how the use of dollar signs ($) controls cell locking when applying formatting across rows (3:47-6:07).
  • Highlight Specific Data: Setting rules to flag values based on criteria, such as schools with more than 35 boys (7:11-7:48) or those with zero teachers (7:50-8:48).
  • Incorporate Logic: Using formulas like SUM within conditional formatting to identify schools with specific social category enrollment (8:51-10:07).

The video emphasizes that conditional formatting is dynamic, meaning the highlights automatically update whenever the underlying data changes (10:35-11:07).
