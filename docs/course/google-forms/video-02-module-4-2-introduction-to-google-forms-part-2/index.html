Module 4 - 2 - Introduction to Google Forms (Part 2)

Jun 6, 2023

This is the second part of the first video in Module 4 - Data Collection with Google Forms. It shows how the form preview can be used when creating a new Google Form.

This video focuses on advanced features for efficient data collection. It covers how to apply response validation (such as ensuring phone numbers are 10 digits or validating email formats), import questions from previous forms to save time (1:46), and organize long surveys by using sections (2:50).

Key highlights include:

  • Data Organization: Moving beyond basic questions by using checkboxes with validation (6:18), creating drop-down menus for long lists (7:05), and reordering questions or options via drag-and-drop (7:41).
  • Form Preview: Using the preview button (8:27) to test the user experience, check required fields, and ensure logical flow between sections.
  • Best Practices: Suggestions on using drop-downs instead of multiple-choice for longer lists to improve accessibility on mobile devices (12:48).
