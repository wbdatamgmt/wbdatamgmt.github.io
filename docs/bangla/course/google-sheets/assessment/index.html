Data Analysis with Google Sheets Assessment

This assessment is optional but highly recommended. Javascript-based answer checking will be added later.

  1. What is the primary use of Google Sheets? a. To create presentations b. To design graphics X. To create and manage spreadsheets d. To edit videos

  2. How can you quickly fill a series of cells with the same data in Google Sheets? X. By using the drag handle (fill handle) from the corner of the cell b. By copying and pasting each cell c. By typing in each cell individually d. By pressing the Enter key repeatedly

  3. Which menu option allows you to create a Pivot Table in Google Sheets? a. Insert > Function X. Data > Pivot table c. View > Pivot table d. Format > Pivot table

  4. What is the purpose of using Conditional Formatting in Google Sheets? a. To change the font size of text X. To automatically format cells based on their values c. To protect the sheet d. To create charts

  5. How can you share a Google Sheet with others? a. By exporting it as a PDF X. By clicking the “Share” button and entering the recipient’s email address c. By printing it d. By copying the data into a Word document

  6. What does the CONCATENATE function do in Google Sheets? X. Combines the contents of multiple cells into one cell b. Counts the number of cells in a range c. Multiplies the values of multiple cells d. Sorts data in ascending order

  7. Which Google Sheets function would you use to count the number of cells in a range that meet a specific condition? a. SUMIF X. COUNTIF c. IF d. AVERAGEIF

  8. Which feature in Google Sheets allows multiple users to work on the same sheet simultaneously? X. Real-time collaboration b. Data validation c. Conditional formatting d. Chart creation

  9. What does the “Protect sheets and ranges” function do in Google Sheets? X. Restricts who can edit certain cells or entire sheets b. Hides rows or columns from view c. Creates a password for the entire spreadsheet d. Deletes protected data automatically

  10. How can you apply a filter to a dataset in Google Sheets? a. By copying the data to a new sheet X. By selecting the data and clicking on the Filter icon in the toolbar c. By sorting the data manually d. By using the SUM function
