Data Visualisation Assessment

Jun 6, 2023

This assessment is optional but highly recommended. Javascript-based answer checking will be added later.

  1. What is the primary purpose of data visualisation? a. To add more complexity to data X. To simplify complex data for easier understanding c. To hide important information d. To make data confusing

  2. Which of the below indicate the importance of data visualisation? (Multiple choice) a. It enables us to see patterns and trends that might not be apparent from looking at the raw data b. It makes data analysis more complicated and time consuming X. It enables us to make data-driven decisions based on facts rather than opinions or assumptions d. It is only useful for experts in data analysis

  3. Why do we add labels to data visualisations? a. To make them look colourful b. To hide information X. To explain what the visualisation means d. To confuse the audience

  4. Which of the following is an advantage of using colour in data visualisation? a. It increases file size b. It makes data harder to interpret X. It helps differentiate categories easily d. It limits creativity in visualisation

  5. Which of these is not a common element of data visualisation? a. Title b. Legend c. Scale X. Eraser

  6. Which type of chart is best suited for showing trends over time? X. Line chart b. Pie chart c. Scatter plot d. Bar chart

  7. Which type of chart should have data that adds up to 100%? a. Line chart X. Pie chart c. Scatter plot d. Histogram

  8. Which type of chart is best suited for comparing values across different categories? a. Scatter plot b. Histogram c. Line chart X. Bar chart

  9. Which one of the following is a good practice for creating data visualisations? a. Trying to say too many things in one chart b. Adding unnecessary design (like 3D chart) c. Not providing labels and data values X. Choosing the right type of chart

  10. Which of the following is NOT a characteristic of effective data visualisation? a. Clarity in presentation b. Consistency in design X. Inclusion of irrelevant data d. Use appropriate scale for the axis
