The participants will learn how to import data from Google Forms into Google Sheets and perform basic data analysis. They will understand how to sort and filter data, use functions like COUNTIF and AVERAGEIF, create pivot tables, create charts and graphs, and use conditional formatting to highlight important data.

## Components

1. Introduction to Google Sheets
2. Sorting and Filtering Data
3. Using Functions in Google Sheets
4. Conditional Formatting
5. Creating Pivot Tables
6. Creating Charts and Graphs
